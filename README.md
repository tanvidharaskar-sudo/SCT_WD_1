# 🌐 Navigation Menu

## 📌 Description
This project is a responsive navigation menu created using HTML, CSS, and JavaScript.  
It helps users easily navigate different sections of a website.

## 🚀 Features
- Responsive design (mobile + desktop)
- Simple and clean layout
- Easy navigation links
- Menu toggle for small screens

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the project
2. Open index.html in your browser

## 📂 Files
- index.html
- style.css
- script.js
